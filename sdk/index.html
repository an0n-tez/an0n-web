One Eleven (SDK) by an0n.tez

More of a library or fxhash template than an SDK, but SDK sounds more official and is great for short urls.

One Eleven (and all canonical installments) are CC0, which means they are public domain and available for anyone to use, with or without attribution.

This SDK makes it really easy for you to bootstrap a fxhash project to use the One Eleven signature piece in another generative work.

Documentation/Examples coming soon, but for now, it can be found on github.

An example of a project using the SDK is One Eleven (Landscape/Landfill).

The unobfuscated source code for (Landscape/Landfill) is also in the github project in the samples folder.
